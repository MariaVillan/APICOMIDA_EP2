# Proyecto: API para Restaurante con Seguridad JWT

## Descripción

Este proyecto es una API para un pequeño restaurante, implementada en Node.js y protegida con JSON Web Tokens (JWT) para la autenticación y autorización de usuarios. Utiliza MongoDB como base de datos.

## Requisitos

- Node.js (v12+)
- npm (v6+)
- MongoDB (v4+)
